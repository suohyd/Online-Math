Section 3.1 Basis and dimension of a vector space

We first define linear independence of vectors and use it to define basis of a vector space. We begin with the following.
Definition 3.1.1. (Linearly dependent vectors).

Let \(V\) be a vector space over a field \(F\text{.}\) A set of vectors \(\{v_1,v_2,\ldots,v_n\}\subset V\) is said to be linearly dependent over \(F\) if there exists scalars \(\alpha_1,\alpha_2,\ldots,\alpha_n\in F\) not all zero such that

\begin{equation*} \alpha_1 v_1+\alpha_2 v_2+\cdots+\alpha_n v_n=0. \end{equation*}

An infinite subset \(S\) of \(V\) is said to be linearly dependent over \(F\) if there is a finite subset of \(S\) which is linearly dependent over \(F\text{.}\)

Definition 3.1.2. (Linearly independent vectors).
A set \(S\) of vectors in \(V\) are said to be linearly independent over \(F\) if \(S\) is not linearly dependent over \(F\text{.}\)
Remark 3.1.3.
A set \(S\) of vectors is linearly independent if and only if each finite subset of \(S\) is linearly independent.
Definition 3.1.4. (Basis of a vector space).
Let \(V\) be a vector space over a field \(F\text{.}\) A basis of \(V\) over \(F\) is a set of linearly independent vectors in \(V\) which spans \(V\) (refer to DefinitionÂ 2.5.13).

Now we define the dimension of a vector space.

Definition 3.1.5. (Dimension of a vector space).

Let \(V\) be a vector space over a field \(F\text{.}\) The cardinality of a basis of \(V\) is called the dimension of \(V\text{.}\) It is denoted by \(\dim_F V\text{.}\)

If \(\dim_F V\) is finite then, \(V\) is said to be a finite-dimensional vector space over \(F\) or if the context is clear then simply finite-dimensional.

If \(\dim_F V\) is infinite then, \(V\) is said to be an infinite dimensional vector space over \(F\) or if the context is clear then simply infinite dimensional.
